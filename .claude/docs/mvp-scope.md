# MVP Scope

첫 번째 MVP의 목표는 재미 검증이다.

핵심 질문:

- NPC 사이에 숨어 있는 도망자를 찾는 것이 재미있는가?
- 술래가 관찰로 도망자를 찾아낼 수 있는가?
- 도망자가 NPC처럼 위장하는 것이 가능한가?
- 미션 실패 시 강제 변신이 긴장감을 만드는가?
- 한 판을 다시 하고 싶게 만드는가?

## Included

MVP에서는 아래 기능만 구현한다.

- 방 생성/입장
- 최소 인원 기반 게임 시작
- 술래(1명)/도망자(N명) 역할 배정
- 3D 맵
- 1인칭 시점 카메라
- NPC 다수 생성 (미션성 행동을 랜덤으로 수행)
- 플레이어 이동
- NPC 자동 이동
- 술래 공격
- 술래 근접 시 도망자 화면 테두리 경고 효과
- NPC 오공격 시 술래 체력 감소
- 도망자 피격 시 탈락
- 간단한 미션 시스템
- 미션 실패 시 5초간 랜덤 생명체로 강제 변신
- 도망자 개인 기여도 점수 (본인에게만 표시)
- 제한 시간
- 상단 HUD (술래/도망자 프로필 + 남은 시간)
- 승패 판정
- 결과 화면
- 탈락자 관전 모드 (관전 대상 선택 가능)
- 참여 티켓(생명 오브젝트) 소모/자동 충전

## Excluded

MVP에서는 아래 기능을 구현하지 않는다.

- 스킨
- 상점
- 랭킹
- 친구 초대
- 복잡한 성장 시스템
- 인앱 결제 (티켓 구매) — 소모/자동 충전 로직 자체는 MVP에 포함하되, 결제 연동은 이후 단계
- 광고
- 고급 AI NPC (경로탐색/추적 등 지능형 AI)
- 대규모 N:N 팀전
- 음성 채팅
- 실시간 채팅
- 길드/클랜
- 복잡한 매치메이킹

## Initial Game Rules

초기 MVP에서는 아래 규칙을 우선 적용한다.

- 한 판은 5분 동안 진행된다.
- 술래는 1명으로 시작한다.
- 도망자는 1명 이상이다.
- 술래 체력은 5개다.
- 술래가 NPC를 공격하면 체력이 1 감소한다.
- 술래가 도망자를 공격하면 해당 도망자는 탈락한다.
- 도망자는 미션을 실패하면 5초 동안 랜덤 생명체로 강제 변신한다.
- 제한 시간이 끝났을 때 도망자가 1명 이상 살아 있으면 도망자 팀이 승리한다.
- 모든 도망자가 탈락하면 술래 팀이 승리한다.
- 술래 체력이 0이 되면 도망자 팀이 승리한다.

## Ticket System (게임 참여 자원)

게임 1회 참여에는 티켓(생명 오브젝트와 유사한 소모형 자원)이 필요하다.

- 최대 보유량은 50개다.
- 게임 1회 참여 시 5개를 소모한다.
- 시간이 지나면 1개씩 자동으로 충전된다 (충전 주기는 구현 시 상수로 분리하고, 기본값은 1시간당 1개로 가정 — 확정 필요 시 재확인).
- 티켓이 부족하면 게임에 참여할 수 없다.
- 티켓 구매(인앱 결제)는 MVP 이후로 미룬다. MVP에서는 소모/자동 충전 로직만 구현한다.
