# Game Concept

이 게임은 1인칭 시점의 2D 멀티플레이 심리 추리 액션 게임이다.

넓은 2D 맵 안에 많은 NPC와 플레이어가 함께 돌아다닌다. 참여자는 NPC처럼 행동하며 술래에게 정체를 들키지 않아야 한다. 술래는 수많은 NPC 사이에서 실제 참여자를 찾아 공격해야 한다.

## Perspective

모든 플레이어(술래, 참여자)는 1인칭 시점으로 게임을 플레이한다.

- 화면에는 자신의 캐릭터 외형이 보이지 않고, 카메라는 플레이어의 시점을 따라간다.
- 술래는 1인칭 시점으로 주변을 둘러보며 NPC와 참여자를 관찰하고 구분해야 한다.
- 참여자는 1인칭 시점에서도 NPC처럼 보이는 행동을 취해야 하므로, 시점 자체가 정체를 숨기는 데 제약이자 전략 요소가 된다.

## Core Idea

NPC 군중 속에 숨어 있는 실제 플레이어를 술래가 찾아내는 게임이다.

## Roles

### Seeker / 술래

- NPC 사이에 숨어 있는 실제 참여자를 찾아 공격한다.
- 참여자를 공격하면 해당 참여자는 탈락한다.
- NPC를 공격하면 술래의 체력이 1 감소한다.
- 술래는 1인당 체력 5개를 가진다.
- 술래의 체력이 모두 떨어지면 술래 팀이 패배한다.

### Hider / 참여자

- NPC처럼 보이고 행동하면서 제한 시간 동안 살아남아야 한다.
- 술래에게 공격당하면 탈락한다.
- 주기적으로 주어지는 미션을 수행해야 한다.
- 미션을 수행하지 못하면 N초 동안 랜덤한 다른 생명체의 모습으로 강제 변신한다.
- 변신 상태에서는 기존 캐릭터와 다른 외형 또는 움직임을 가지므로 술래에게 정체가 들킬 위험이 커진다.
- 변신 시간이 끝나면 원래 모습으로 돌아온다.

### NPC

- 맵 안을 자동으로 돌아다니는 비플레이어 캐릭터다.
- 참여자가 숨어들 수 있는 군중을 만든다.
- 술래의 판단을 어렵게 만든다.
- 참여자의 행동 위장을 돕는다.

## Mission System

게임 중간중간 참여자에게 미션이 주어진다.

미션 예시:

- 오른쪽으로 달리는 모션을 취하기
- 점프하기
- 특정 방향으로 이동하기
- 잠시 멈추기
- 특정 오브젝트 근처로 이동하기
- 특정 감정표현 또는 행동 수행하기

미션 규칙:

- 미션은 일정 주기마다 발생한다.
- 참여자는 제한 시간 안에 미션을 수행해야 한다.
- 미션을 수행하지 못하면 N초 동안 랜덤한 다른 생명체의 모습으로 강제 변신한다.
- 강제 변신은 참여자의 정체가 드러날 수 있는 위험 상태로 작동한다.
- 변신 중에도 이동과 기본 행동은 가능하다.
- 변신 시간이 끝나면 원래 캐릭터 모습으로 돌아온다.

## Win Conditions

게임은 제한 시간 N분 동안 진행된다.

- 참여자 팀은 제한 시간이 끝났을 때 생존자가 남아 있거나, 술래의 체력이 모두 소진되면 승리한다.
- 술래 팀은 제한 시간 안에 참여자를 모두 찾아 제거하면 승리한다.
- N:N 팀전에서는 게임 종료 시점에 생존자 수, 제거 수, 팀 점수 등을 기준으로 승패를 결정한다.

## Match Flow

1. 유저가 방에 입장한다.
2. 최소 인원이 충족되면 게임을 시작할 수 있다.
3. 각 유저에게 역할이 배정된다.
4. 맵에 NPC가 생성된다.
5. 참여자는 NPC 사이에 섞여 이동한다.
6. 술래는 참여자로 의심되는 캐릭터를 공격한다.
7. 참여자에게 주기적으로 미션이 주어진다.
8. 탈락자는 관전 상태로 전환된다.
9. 제한 시간이 끝나거나 승리 조건이 충족되면 게임이 종료된다.
10. 결과 화면에서 승리 팀, 생존자 수, 술래 체력, 개인 기여도를 보여준다.
